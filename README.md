# Network Stability Logger

A Go project for monitoring and logging network stability metrics.

## Getting Started

### Prerequisites
- Go 1.21 or higher

### Installation

```bash
go mod download
```

### Running

```bash
go run main.go
```

### Building

```bash
go build -o network-stability-logger
```

## Project Structure

- `main.go` - Entry point
- `go.mod` - Module definition
